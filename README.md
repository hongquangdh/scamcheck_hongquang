ScamCheck là 1 dự án 1 thành viên (và 1 AI) phục vụ mục đích kiểm tra scam thuận tiện.  
Đây là 1 dự án cho Trại hè FCT 2026 do công ty FPT Software tổ chức.  
Support .png, .jpg and .webp  

Disclaimer: Các kết quả không thể đúng 100%, có thể bị vượt qua bởi các hình thức lừa đảo tinh vi hơn.

Và đây là layout của web:
```
├── app.py (backend, chắc chắn rồi)
├── index.html (để chạy web)
├── requirement.txt (các model cần thiết)
├── templates/
│   └── index.html (để chạy local)
├── static/
│   ├── app.js (để support frontend)
│   └── style.css (layout trang web)
├── data/
│   ├── hotlines.json (chưa update hotlines)
│   └── scam_types.json (các kiểu scam phổ biến)
├── Procfile (chạy gunicorn)
└── README.md (README file, obviously xD )
```
